# MMORPG Telegram WebApp

Простая MMORPG игра для Telegram WebApp с системой перемещения по чанкам, боем, крафтом и другими механиками.

## Структура проекта

```
.
├── index.html          # Главный HTML файл с UI
├── js/
│   ├── init.js         # Инициализация игры и Telegram WebApp
│   ├── gameState.js    # Управление состоянием игры и сохранениями
│   ├── gameData.js     # Константы игры (локации, враги, предметы и т.д.)
│   ├── movement.js     # Система перемещения по чанкам и быстрого перемещения
│   ├── map.js          # Рендеринг карты локации
│   ├── events.js       # Генерация событий в чанках
│   ├── gathering.js    # Система сбора ресурсов
│   ├── combat.js       # Система боя
│   ├── inventory.js    # Управление инвентарём и экипировкой
│   ├── locations.js    # Список локаций и быстрое перемещение
│   ├── city.js         # Город: крафт, банк, аукцион
│   ├── mastery.js      # Система мастерства
│   └── ui.js           # UI функции (навигация, заголовок)
└── README.md           # Этот файл
```

## Система перемещения

### Основные изменения:
1. **Чанки вместо узлов**: Локации теперь разделены на чанки (например, 10, 15, 20 чанков)
2. **Линейное движение**: Одна кнопка "Сделать шаг" для движения вперёд по чанкам
3. **Конец локации**: На последнем чанке можно начать локацию заново или перейти в следующую
4. **Открытие локаций**: Локации открываются при первом переходе в них
5. **Быстрое перемещение**: Из карты можно быстро переместиться в открытую локацию (время = сумма чанков всех промежуточных локаций)

## Развёртывание на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы в репозиторий
3. В настройках репозитория включите GitHub Pages для папки `/root`
4. Используйте URL вида: `https://username.github.io/repository-name/`

## Использование с Telegram Bot

1. Создайте бота через @BotFather
2. Используйте команду `/newapp` для создания WebApp
3. Укажите URL вашего GitHub Pages сайта
4. Бот будет открывать игру в Telegram

## Разработка

Код разбит на модули ES6 для удобства разработки и поддержки. Все модули находятся в папке `js/`.

### Основные модули:
- `gameState.js` - управление состоянием игры (S) и сохранениями
- `gameData.js` - все константы игры (локации, враги, предметы)
- `movement.js` - логика перемещения по чанкам
- `combat.js` - система боя
- `city.js` - функции города (крафт, банк, аукцион)
